# RtPbrSurvey Requests

This directory tracks requests from Tank Physics Sandbox to the RtPbrSurvey project.

Use this area for renderer-side requirements that are discovered from the Tank host side. Keep the request text mostly in English so it can be copied into RtPbrSurvey tasks or issues. Keep a short Japanese summary list for quick project review.

## Files

- `requests.md`: English request details.
- `summary.ja.md`: Japanese summary list.
